## Installation

### 1. Download or Clone the Repository

To get started with the extension, download or clone this repository to your local machine:

```bash
git clone https://github.com/amangh30/whatsapp-sticker-extension.git
```

---

### 2. Load the Extension in Chrome

1. Open **Google Chrome**.
2. Navigate to `chrome://extensions/` in the address bar.
3. Toggle **Developer mode** on (top-right corner).
4. Click **Load unpacked**.
5. Select the folder where you downloaded or cloned the repository.

---

### 3. Enable the Extension

After loading the extension, it will appear in your Chrome extensions list.
You may need to:

* Pin the extension icon from the Chrome toolbar for easy access.
* Click the extension icon to activate it or adjust its settings.

---

### 4. Using the Extension in Other Browsers

If you want to use this extension in browsers like **Firefox**, **Brave**, **Edge**, or **Opera**, the process is similar:

* Open the browser's extensions page.
* Enable **Developer mode** or **Load Unpacked** mode (name varies by browser).
* Select the cloned/downloaded folder.
