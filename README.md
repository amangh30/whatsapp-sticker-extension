## Installation

### 1. Download or Clone the Repository

To get started with the extension, download or clone this repository to your local machine.

```
git clone https://github.com/amangh30/whatsapp-sticker-extension.git
```

### 2. Load the Extension in Chrome

1. Open Google Chrome.
2. Go to `chrome://extensions/` in the address bar.
3. Toggle **Developer mode** on (located in the top-right corner).
4. Click **Load unpacked**.
5. Select the folder where you downloaded/cloned the repository.

### 3. Enable the Extension

Once you've loaded the extension, it should be visible in your Chrome extensions list. You may need to click the extension icon to enable it or interact with it/
